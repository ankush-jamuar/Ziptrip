## Assignment Overview

This repository contains the solutions for the Ziptrrip Developer Assignment.

The assignment consists of two challenges:

### Challenge 1

JavaScript and CSS problem-solving questions covering:

* Pattern Printing
* String Reversal
* Removing Duplicates from an Array
* CSS Selectors
* CSS Layout Techniques

Multiple approaches have been provided wherever requested.

### Challenge 2

A Full Stack Todo Application built using:

#### Frontend

* React
* React Router
* Axios

#### Backend

* Node.js
* Express.js

#### Storage

* JSON File Storage
